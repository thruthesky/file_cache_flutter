# file_cache_flutter

Flutter 애플리케이션을 위한 파일 캐시 라이브러리입니다. 파일을 쉽게 캐시하여 성능을 향상시키고 네트워크 사용량을 줄일 수 있습니다.

## 기능

- 네트워크에서 다운로드한 파일을 로컬에 캐시
- 캐시된 파일 자동 관리
- 간편한 API

## 설치

`pubspec.yaml` 파일에 다음을 추가하세요:

```yaml
dependencies:
  file_cache_flutter: ^0.0.1
```

그런 다음 패키지를 설치합니다:

```bash
flutter pub get
```

## 사용법

```dart
import 'package:file_cache_flutter/file_cache.dart';

// 사용 예제는 추후 추가 예정
```
